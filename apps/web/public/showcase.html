Open-source living sheet music

Sheet music that plays, slows down, and syncs to real recordings.

An open alternative to Soundslice, built around a portable file format instead of a hosted service. Practice with the tools that matter, then embed any piece on your own site.

Try it live View on GitHub

See it embedded

This is a real OpenVoicing player, embedded on this page with one line of HTML.

Practice, not just playback

Slow a passage down without changing pitch, loop it, add a count-in, and follow a metronome.

Sync to real recordings

Line a real performance up with the score so playback follows the notation, bar by bar.

Own your files

Every piece exports as a self-contained .ovb bundle. No account, no lock-in — host it anywhere.

Editable notation

Import MusicXML or Guitar Pro, edit the notes, and export. It's a real editor, not a viewer.

Embed a piece in one line

Host a bundle anywhere, drop in the script, and point a div at it.

<script src="https://your-host/openvoicing-embed.js"></script>
<div data-openvoicing-bundle="https://example.com/tune.ovb"></div>

Full options, methods, and events are in the embed API reference.
