# Internal drafts and design artifacts

This folder holds **working notes**, not shipped documentation. Nothing here is
authoritative: designs may be superseded, and the council-consensus JSON files
are raw outputs of a design exercise, kept for provenance.

For current, maintained documentation start at [`../index.md`](../index.md).

Contents:

- `full-fidelity-model-plan*.md`, iterations of the score-model design. The
  latest thinking that has actually shipped lives in the code and in
  [`../architecture.md`](../architecture.md).
- `feature-council-consensus.json`, `ux-council-consensus.json`, outputs of
  multi-perspective design reviews used to prioritize work.

The project roadmap is [`../../PLAN.md`](../../PLAN.md).
